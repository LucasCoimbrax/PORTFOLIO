Clone da Netflix - OtakuMania

Sobre

Este projeto é um clone do Netlix, onde temos os animes nos lugares de filmes e séries. Tem-se um anime em destaque com seu nome, uma sinopse e botões "Assistir Agora" e "Informações", que redirecionam o usuário para seus designados lugares. Na parte inferior da página, contamos com um carrossel que se movimenta com o mouse e mostra outros animes. Este projeto foi desenvolvido com o intuito de aplicar os conhecimento adquiridos de HTML, CSS e JavaScript até o momento.
